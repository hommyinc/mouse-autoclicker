# Mouse-Autoclicker

- 이 레포지토리는 카카오톡 프로필 공감 스티커를 누르는 등 개쓸데없는 짓을 하기 위해 만들어졌습니다.
- 물론 [오토클릭](https://autoclick.co.kr/)과 같이 더욱 멋지고 편리한 기능이 많은 프로그램이 이미 있어, 이 레포지토리의 가치는 전무합니다.

#### 사용법

1. 가상환경에 설치
```sh
pip install -r requirements.txt
```

2. 설정값 조절: [config.py](./config.py)

3. 사용
```python
python run.py
```
